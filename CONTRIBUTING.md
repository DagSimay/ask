# Contributing

## How to Contribute

1. Fork this repository.
2. Create a new branch: git checkout -b feature/your-feature
3. Make your changes.
4. Commit with a clear message: git commit -m "describe your change"
5. Push and open a Pull Request.

## Guidelines

- Test with all three input methods (args, pipe, both combined).
- Only use curl and jq as dependencies.
- Document any new usage in the README.
